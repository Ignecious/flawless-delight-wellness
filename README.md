# Flawless Delight Wellness - Landing Page

Mock UI for the Flawless Delight Wellness digital booking platform built with Angular 17.

## Tech Stack
- Angular 17.3.x
- TypeScript 5.2.x
- SCSS
- Standalone Components

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   ng serve
   ```

3. Open browser to `http://localhost:4200`

## Features
- ✅ Responsive landing page
- ✅ Sage green wellness theme
- ✅ Signature services section
- ✅ Top treatments showcase
- ✅ Personalized skincare section
- ✅ Mobile-friendly design
